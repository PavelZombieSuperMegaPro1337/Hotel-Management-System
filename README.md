# Hotel-Management-System
цель
Создать консольную программу для автоматизации работы администратора отеля: учёт номеров, бронирование, заселение/выселение гостей, создание отчётов.


функциональные требования
1. Управление номерами:
добавление/редактирование номеров (номер, тип, вместимость, цена за ночь, статус: свободен/занят/на уборке);
просмотр списка всех номеров с фильтрацией по статусу/типу.

2. Бронирование:
резервирование номеров (гость, номер, даты заезда/выезда, количество гостей);
проверка доступности номеров;
отмена брони.

3. Заселение/выселение:
регистрация гостя при заселении (ФИО, контакты, данные паспорта);
расчёт итоговой цены (с учётом дней проживания и всех доп услуг);
оформление выселения.

4. Дополнительные услуги:
заказ завтрака в номер;
вызов такси;
бронирование трансфера;
запись на спа‑процедуры.

5. Отчёты и аналитика:
список занятых номеров на сегодня;
прогноз загрузки на неделю или месяц;
доход за период;
самые популярные виды номеров;
статистика по доп услугам.

7. Работа с данными:
сохранение всей информации в файл;
загрузка данных при запуске программы;

резервное копирование.
